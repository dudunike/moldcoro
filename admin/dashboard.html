⚠️ Configuração necessária: permissões do banco de dados

Execute este SQL no Supabase SQL Editor para liberar o acesso do painel:

ALTER TABLE public.chaves_events ENABLE ROW LEVEL SECURITY;

DROP POLICY IF EXISTS "anon_insert_chaves"  ON public.chaves_events;
CREATE POLICY "anon_insert_chaves"  ON public.chaves_events FOR INSERT TO anon      WITH CHECK (true);

DROP POLICY IF EXISTS "auth_select_chaves"  ON public.chaves_events;
CREATE POLICY "auth_select_chaves"  ON public.chaves_events FOR SELECT TO authenticated USING (true);

DROP POLICY IF EXISTS "auth_delete_chaves"  ON public.chaves_events;
CREATE POLICY "auth_delete_chaves"  ON public.chaves_events FOR DELETE TO authenticated USING (true);

Após executar o SQL, clique em ↺ Atualizar no topo.
